# Prune (GET)

| Method | Description             |
| ------ | ----------------------- |
| GET    | Prunes a specified fork |

#### Request URI

```bash
/v1/chain/tips/prune/{hash}
```

#### Parameters Used

| ID   | TYPE   | Description     |
| ---- | ------ | --------------- |
| hash | STRING | The header hash |

#### Response

```bash
OK status code http response will be returned with no reply.
```
